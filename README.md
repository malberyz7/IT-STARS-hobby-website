# Hobby Inspiration Website

A modern, responsive school competition website built with Next.js App Router, Tailwind CSS, and Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech stack

- Next.js 14 (App Router)
- React 18
- Tailwind CSS
- Framer Motion
- Reusable UI components inspired by shadcn/ui
